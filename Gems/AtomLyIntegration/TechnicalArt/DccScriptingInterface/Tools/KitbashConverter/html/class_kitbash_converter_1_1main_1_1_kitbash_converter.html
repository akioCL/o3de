◆ set_relative_path_clicked()

def KitbashConverter.main.KitbashConverter.set_relative_path_clicked (   self)

This button action keeps output file locations positioned in the best accessible locations.

If the user attempts to assign the relative output path to a path thats base is NOT the input directory, a warning dialog box will prompt to select a compatible path. The path that is set by default ('Objects/KB3D/CityKit') gives the recommended formatted location.

◆ start_maya_session()

def KitbashConverter.main.KitbashConverter.start_maya_session (   self,
  target_fbx 
)

This starts the exchange between the standalone QT UI and Maya Standalone to process FBX files.

The information sent to Maya is the FBX file for processing, as well as the base directory and relative destination paths. There is a pipe that can be used to shuttle processed information back to the script, but at this stage it is not being used.

Parameters
target_fbxThe file to be sent to Maya for processing
